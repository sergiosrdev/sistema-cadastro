# Sistema de Cadastro

Sistema completo de cadastro desenvolvido com React (frontend) e Node.js/Express (backend).

## Funcionalidades

- ✅ Cadastro de usuários
- ✅ Listagem de usuários
- ✅ Edição de usuários
- ✅ Exclusão de usuários
- ✅ Validação de formulários
- ✅ Interface responsiva
- ✅ Mensagens de feedback

## Tecnologias

### Frontend
- React 18
- TailwindCSS
- Axios
- React Router

### Backend
- Node.js
- Express
- SQLite3
- CORS
- Nodemon

## Instalação

1. Clone o repositório
2. Execute o comando para instalar todas as dependências:
```bash
npm run install-all
```

## Como Executar

Para desenvolvimento (inicia frontend e backend simultaneamente):
```bash
npm run dev
```

Acesse:
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000
